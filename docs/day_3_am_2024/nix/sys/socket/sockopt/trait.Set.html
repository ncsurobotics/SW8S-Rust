Helper trait that describes what is expected from a SetSockOpt setter.

fn ffi_ptr(&self) -> *const c_void

Returns a pointer to the stored value. This pointer will be passed to the system’s setsockopt call (man 3p setsockopt, argument option_value).

source

fn ffi_len(&self) -> socklen_t

Returns length of the stored value. This pointer will be passed to the system’s setsockopt call (man 3p setsockopt, argument option_len).
