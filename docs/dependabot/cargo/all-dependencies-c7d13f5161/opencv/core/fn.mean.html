pub fn mean(src: &impl ToInputArray, mask: &impl ToInputArray) -> Result<Scalar>
Expand description

Calculates an average (mean) of array elements.

The function cv::mean calculates the mean value M of array elements, independently for each channel, and return it: When all the mask elements are 0’s, the function returns Scalar::all(0)

§Parameters

  • src: input array that should have from 1 to 4 channels so that the result can be stored in Scalar_ .
  • mask: optional operation mask.

§See also

countNonZero, meanStdDev, norm, minMaxLoc

§C++ default parameters
