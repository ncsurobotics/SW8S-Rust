pub fn get_structuring_element_def(shape: i32, ksize: Size) -> Result<Mat>
Expand description

Returns a structuring element of the specified size and shape for morphological operations.

The function constructs and returns the structuring element that can be further passed to #erode, dilate or #morphologyEx. But you can also construct an arbitrary binary mask yourself and use it as the structuring element.

§Parameters

  • shape: Element shape that could be one of [morph_shapes]
  • ksize: Size of the structuring element.
  • anchor: Anchor position within the element. The default value means that the anchor is at the center. Note that only the shape of a cross-shaped element depends on the anchor position. In other cases the anchor just regulates how much the result of the morphological operation is shifted.

§Note

This alternative version of get_structuring_element function uses the following default values for its arguments:

  • anchor: Point(-1,-1)
