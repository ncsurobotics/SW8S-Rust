fn forward_pos2(mask1: i32, mask2: i32) -> usize
Expand description

Compute the position of the first matching byte from the given masks. The position returned is always in the range [0, 15]. Each mask corresponds to the equality comparison of a single byte.

The masks given are expected to be the result of _mm_movemask_epi8, where at least one of the masks is non-zero (i.e., indicates a match).
